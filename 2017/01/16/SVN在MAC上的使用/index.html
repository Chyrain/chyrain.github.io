在MAC上使用SVN

SVN在MAC

  • checkout一个远程库
  • 从本地导入代码到服务器(第一次初始化导入)，在终端中输入
  • 提交代码到服务器：svn commit -m “修改了main.m文件”
  • 更新服务器端的代码到客户端：svn update
  • svn的其他用法，可以在终端输入：svn help
